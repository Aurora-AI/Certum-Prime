# Neurodesign Aplicado à Experiência Financeira (Projeto Aurora)

Este skill fornece diretrizes completas para a criação de experiências digitais financeiras que ativam confiança, segurança e decisão no cérebro humano, baseadas no documento "Neurodesign Aplicado à Experiência Financeira".

## 1. Fundação Teórica: Os 4 Pilares Cognitivos

O design deve ativar regiões específicas do cérebro para reduzir a fricção e aumentar a conversão.

1.  **Segurança (Sistema Límbico)**
    *   *Gatilho:* Previsibilidade, controle e feedback imediato.
    *   *Aplicação:* Feedback visual para toda ação (<100ms), termos claros, "Glassmorphism" (transparência = nada a esconder), bordas luminosas ("Cofre").
    *   *Cores:* Azul profundo (confiança), Verde (estabilidade).

2.  **Família/Pertencimento (Neurônios-espelho)**
    *   *Gatilho:* Familiaridade, conexão e empatia.
    *   *Aplicação:* Abstração conectiva (partículas orbitando um núcleo), padrões reconhecíveis (Lei de Jakob), cópia afetiva ("Proteja quem você ama").

3.  **Sucesso/Poder (Sistema de Recompensa - Dopamina/Testosterona)**
    *   *Gatilho:* Superação de obstáculos, status, agência.
    *   *Aplicação:* "UX Soberana" (o site obedece ao usuário), estética "Dark Mode" de luxo, detalhes em Ouro Líquido/Platina, interações que "levitam" ao hover.

4.  **Poder/Controle (Córtex Pré-Frontal)**
    *   *Gatilho:* Escolha informada e validação lógica.
    *   *Aplicação:* Simuladores interativos (UI Generativa), dados claros, gráficos em tempo real, ausência de "Vale da Estranheza" (usar inteligência abstrata em vez de avatares humanos falsos).

## 2. Filosofia de Design: "Cristal Holográfico" & "UX Soberana"

*   **Metáfora Visual:** "Cristal Holográfico" — materialidade tátil do vidro (peso, refração) + luminescência etérea (futuro, dados).
*   **UX Soberana:** A interface antecipa a intenção. O cursor é magnético. O fundo (fluido neural) reage à presença do mouse. O site não é um labirinto, é um concierge.
*   **Paleta:** Fundo "Void" (#050505), Azul Bioluminescente (#00C2FF), Roxo Profundo (#4A00E0), Ouro Líquido (#D4AF37).
*   **Tipografia:** Sans-Serif Geométrica (Inter, Roobert) para dados + Serif de Alto Contraste (Ogg, Canela) para títulos.

## 3. Stack Tecnológica "Aurora"

Para atingir a nota 9.0+ no Awwwards e performance máxima:

*   **Core:** Next.js 15 (App Router) + TypeScript.
*   **3D/WebGL:** React Three Fiber (R3F) + Drei (MeshTransmissionMaterial para vidro real).
*   **Animação:** Framer Motion (física de mola/spring).
*   **Estilo:** Tailwind CSS + CSS Modules.
*   **UI Primitivos:** shadcn/ui (Radix UI).
*   **IA/Agentes:** Vercel AI SDK (Generative UI) + CrewAI (Backend Agents).
*   **Dados:** Supabase (PostgreSQL + pgvector).

## 4. Estratégia de Conteúdo e SEO (GEO)

Otimização para **Motores Generativos (GEO - Generative Engine Optimization)**:

*   **Entidades, não palavras-chave:** Foco em Schemas ricos (FinancialProduct, Organization, FAQPage).
*   **Knowledge Graph:** "Citações Contextuais" e "Respostas Atômicas" (fatos densos para LLMs citarem).
*   **Estrutura Soberana:** `/llm-manifest.json` ou sitemap estruturado para guiar crawlers de IA à "Verdade" da marca.

## 5. Agentes do Sistema (CrewAI)

### Construtores (Dev)
*   **O Arquiteto (Sistema 2):** Garante qualidade de código e arquitetura. Bloqueia CLS e scripts pesados.
*   **O Neuro-Designer (Sistema 1):** Analisa regressão visual, contraste e fluidez (60fps).

### Guardiões (Runtime)
*   **O Ademilover (Vendas):** Consultor via Chat/Concierge. Metodologia S.R.V. (Sonho, Realidade, Velocidade).
*   **A Sentinela SEO:** Monitora tendências em Perplexity/Gemini e atualiza Schemas.
*   **O Alquimista de Dados:** Atualiza índices financeiros (Selic, CDI) para os simuladores.

## 6. Plano de Execução (Sprints de 10 Dias)

*   **Dia 1:** Imersão Cognitiva e Benchmarking.
*   **Dia 2:** Arquitetura de Informação e SEO Semântico.
*   **Dia 3:** Prototipagem de Alta Fidelidade (Fluxos de Simulação).
*   **Dia 4:** Setup do Sistema Base (Next.js, Cores, Tipografia).
*   **Dia 5:** Camada de Resposta (Background WebGL/Canvas).
*   **Dia 6:** Páginas-Chave (Landing, Família - "Momentos de Poder").
*   **Dia 7:** Ferramentas Interativas (Simuladores, Planejador com Recharts).
*   **Dia 8:** Integração, Formulários e SEO Técnico.
*   **Dia 9:** Testes de Usabilidade e Ajustes de Fricção.
*   **Dia 10:** Polimento, Documentação e Deploy.

## 7. Instruções para Agentes

Ao utilizar esta skill:
1.  **Priorize a Performance Cognitiva:** Cada interação deve ter feedback <100ms.
2.  **Valide via Neurodesign:** Antes de finalizar uma tela, verifique se ela atende aos gatilhos de Segurança, Família ou Poder.
3.  **Use Generative UI:** Prefira interfaces que se montam dinamicamente (ferramentas no chat) a formulários estáticos longos.
4.  **Mantenha a Estética "Aurora":** Siga rigorosamente a paleta e a filosofia do "Cristal Holográfico".
