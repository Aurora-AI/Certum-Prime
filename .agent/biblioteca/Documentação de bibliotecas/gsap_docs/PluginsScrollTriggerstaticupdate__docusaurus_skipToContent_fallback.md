Checks where the scrollbar is and updates all ScrollTrigger instances'
progress
and

Performs the following actions for
ALL
ScrollTrigger instances:
Checks the location of the scrollbar and updates each ScrollTrigger's
progress
and
direction
values accordingly.
Controls the linked animation (if necessary).
Fires the appropriate callbacks like onEnter, onToggle, onUpdate, etc.
ScrollTrigger.update()
does NOT recalculate the start/end positions according to any DOM changes you've made.
That's what
ScrollTrigger.refresh()
is for.
