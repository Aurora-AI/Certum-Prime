[​](#conformidade) Conformidade
-------------------------------

### [​](#conformidade-em-saúde-baa) Conformidade em saúde (BAA)

Se um cliente tem um Business Associate Agreement (BAA) conosco e deseja usar Claude Code, o BAA será automaticamente estendido para cobrir Claude Code se o cliente tiver executado um BAA e tiver Zero Data Retention (ZDR) ativado. O BAA será aplicável ao tráfego de API desse cliente fluindo através do Claude Code.
